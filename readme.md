pip install -r requirements.txt

To run the web app :
Step 1 : First select the file `app.py` because the commands below will be run upon it.
Step 2 : Press Ctrl + J
Step 3 : Type cd (space) code
Step 4 : Press Enter
Step 5 : Type python (space) app.py
Step 6 : Press Enter
Step 7 : Click on the 'Follow Link'
