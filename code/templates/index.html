How To Use ImageGuard?

Firstly, you need to sign-up if you are new to ImageGuard. If you already have an account, enjoy then!!! ImageGuard is free to use. You can check results of an image by uploading it. For this, click on Upload button on the Navigation bar. It will take you to the Upload page. As soon as it gets uploaded, you can see the results by clicking on the File List button & then on the View link of that image.

Extensions Allowed

The uploading as well as the detection of authentication of only these given file extensions are allowed: '.jpg', '.jpeg' & '.png'
